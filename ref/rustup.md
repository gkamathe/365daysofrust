## rustup usage

```
$ rustup show

$ rustup default
$ rustup default nightly

$ rustup target list

$ rustup toolchain uninstall nightly

$ rustup toolchain help

$ rustup man cargo

$ rustup check

$ rustup update

$ rustup which rustc

$ rustup which cargo

$ rustup doc
$ rustup doc --book
$ rustup doc --std
$ rustup doc --cargo
```
